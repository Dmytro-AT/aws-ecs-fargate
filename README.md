# AWS ECS FARGATE
Lite version - without CloudFront and Route53 and ASM
 - Frontend - static content on S3 bucket
 - Backend - one container AWS ECS FARGATE + ALB + SG
   - .env  - SSM (AWS Systems Manager > Parameter Store)
